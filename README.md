# mgpeb-aurora-siger
Protótipo em Python do Módulo de Gerenciamento de Pouso e Estabilização de Base (MGPEB) da missão Aurora Siger. Implementa fila de pouso, listas e pilha de histórico, regras de decisão com lógica booleana (autorizado/espera/alerta/emergência), buscas lineares e ordenações simples (bubble/selection), além de simulações de telemetria para testes
